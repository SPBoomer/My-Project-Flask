# 🚀 Flask Visit Logger — Веб-приложение на Python

Простое и стильное веб-приложение для регистрации посещений. Разработано в рамках производственной практики по ПМ.01 «Разработка модулей программного обеспечения для компьютерных систем».

## ✨ Особенности
*   **Backend:** Python + Flask (микрофреймворк).
*   **Frontend:** Адаптивная верстка, Dark Mode дизайн, CSS Variables.
*   **Database:** SQLite с использованием SQLAlchemy (ORM).
*   **Templates:** Шаблонизатор Jinja2 для динамического отображения данных.

---

## 🛠 Технологический стек
*   **Язык:** Python 3.12+
*   **Web Framework:** Flask
*   **ORM:** Flask-SQLAlchemy
*   **Интерфейс:** HTML5, CSS3 (Modern UI)

---

## 🚀 Инструкция по запуску

### 1. Клонирование репозитория
```bash
git clone https://github.com/SPBoomer/My-Project-Flask
cd My-Project-Flask
```

### 2. Установка зависимостей
```bash
pip install Flask Flask-SQLAlchemy

```
### 3. Запуск приложения
```bash
python app.py
```
## 📂 Структура проекта
**app.py** — Главный файл приложения (логика и база данных).
**templates/** — HTML-шаблоны страниц.
**static/** — Статические файлы (стили CSS).
**database.db** — Локальная база данных SQLite (создается автоматически).
